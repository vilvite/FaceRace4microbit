# FaceRace4microbit
Teachable Machine webcam classifier that connects to micro:bit and sends predictions over usb serial or Bluetooth.


[![Face Race](./icon/fr512.png)](https://vilvite.github.io/FaceRace4microbit/facerace.html)

[Face Race / KI Kart](https://vilvite.github.io/FaceRace4microbit/facerace.html) - Reads tm's as downloaded zip-files. Installable for offline usage. Android friendly.


[Github source code](https://github.com/vilvite/FaceRace4microbit/)

[BitBotRemote](https://vilvite.github.io/FaceRace4microbit/bitbotremote.html) - Android friendly universal micro:bit remote with gamepad support
